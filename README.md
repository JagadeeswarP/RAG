        <h1>RAG Q&amp;A App using Gemini AI</h1>
        <p><strong>Built with:</strong> LangChain, FAISS, and Google Gemini</p>
        <h2>Overview</h2>
        <p>
            This app allows you to interact with various types of documents and web content
            to ask questions and get answers using the power of Google Gemini AI.
        </p>
        <p>The app supports multiple input types, including:</p>
        <ul>
            <li>Link (URL)</li>
            <li>PDF</li>
            <li>Text</li>
            <li>DOCX</li>
            <li>TXT</li>
        </ul>
        <h2>Features</h2>
        <ul>
            <li>Upload documents or provide links to process and store information.</li>
            <li>Text embeddings are generated using <strong>Google Gemini</strong> for better context understanding.</li>
            <li>Ask questions based on the uploaded content and receive answers using the Gemini model.</li>
        </ul>
        <h2>Setup</h2>
        <p>Follow these steps to get the app running locally:</p>
        <h3>1. Clone the repository</h3>
        <pre><code>git clone https://github.com/yourusername/your-repository-name.git</code></pre>
        <h3>2. Navigate to the project directory</h3>
        <pre><code>cd your-repository-name</code></pre>
        <h3>3. Create a virtual environment (optional but recommended)</h3>
        <pre><code>python -m venv venv</code></pre>
        <h3>4. Activate the virtual environment</h3>
        <p><strong>Windows:</strong></p>
        <pre><code>venv\Scripts\activate</code></pre>
        <p><strong>macOS/Linux:</strong></p>
        <pre><code>source venv/bin/activate</code></pre>
        <h3>5. Install dependencies</h3>
        <pre><code>pip install -r requirements.txt</code></pre>
        <h3>6. Create a <code>.env</code> file</h3>
        <pre><code>GOOGLE_API_KEY=your_api_key_here</code></pre>
        <h3>7. Run the app</h3>
        <pre><code>streamlit run app.py</code></pre>
        <h2>How to Use</h2>
        <ol>
            <li>Choose an input type (Link, PDF, Text, DOCX, TXT).</li>
            <li>Provide the necessary input:
                <ul>
                    <li>For Links, paste the URL(s).</li>
                    <li>For PDFs/DOCX/TXT, upload the respective file(s).</li>
                    <li>For Text, paste the content directly.</li>
                </ul>
            </li>
            <li>Click the <strong>Process Input</strong> button to convert the input into a vector store.</li>
            <li>Once the vector store is created, type your question in the text input box.</li>
            <li>Click <strong>Submit Question</strong> to get an answer generated by the app.</li>
        </ol>
        <h2>Notes</h2>
        <ul>
            <li>This app uses the <strong>Google Gemini API</strong> to generate embeddings and answers.</li>
            <li>Ensure you have a valid API key and that your environment variables are set correctly.</li>
        </ul>
